# UART-Hamming-Oversampling

## Overview

This project implements a reliable UART communication system using 16× oversampling and Hamming (12,8) error correction in Verilog HDL.

## Features

* UART-based serial communication
* Hamming (12,8) encoding and decoding
* Single-bit error detection and correction
* 16× oversampling receiver
* Noise and glitch tolerance
* Verilog HDL implementation
* Vivado simulation and verification

## Project Structure

* src/ – Verilog source files
* simulation/ – Testbench and simulation files
* images/ – Waveforms, synthesis and timing results
* reports/ – Project documentation
